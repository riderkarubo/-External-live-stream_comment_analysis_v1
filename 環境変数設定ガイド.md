# 環境変数設定ガイド

## .envファイルの編集方法

### 1. テキストエディタで開く

CursorやVS Codeなどのエディタで`.env`ファイルを開いて編集してください。

### 2. 設定すべき値

#### Anthropic Claude APIキーの設定

1. [Anthropic Console](https://console.anthropic.com/)にアクセス
2. アカウントにログイン
3. API Keysセクションで新しいAPIキーを作成（または既存のキーをコピー）
4. `.env`ファイルで`your_anthropic_api_key_here`を実際のAPIキーに置き換え

```env
ANTHROPIC_API_KEY=sk-ant-api03-実際のAPIキーをここに入力
```

#### Google Sheets API設定

**Option 1: サービスアカウントJSONファイルを使用（推奨）**

1. [Google Cloud Console](https://console.cloud.google.com/)にアクセス
2. 新しいプロジェクトを作成（または既存のプロジェクトを選択）
3. 「APIとサービス」→「ライブラリ」から「Google Sheets API」を有効化
4. 「APIとサービス」→「認証情報」→「認証情報を作成」→「サービスアカウント」
5. サービスアカウント名を入力して作成
6. 作成したサービスアカウントをクリック→「キー」タブ→「キーを追加」→「JSON」を選択
7. ダウンロードしたJSONファイルをプロジェクトフォルダに保存（例: `service-account.json`）
8. `.env`ファイルでパスを設定

```env
GOOGLE_SERVICE_ACCOUNT_FILE=./service-account.json
# または絶対パス
# GOOGLE_SERVICE_ACCOUNT_FILE=/Users/issy_s/コーディング/live-stream_comment_analysis/service-account.json
```

**Option 2: JSONを環境変数として設定**

サービスアカウントJSONファイルの中身をそのまま環境変数に設定することもできます（あまり推奨されません）。

```env
GOOGLE_CREDENTIALS_JSON={"type": "service_account", "project_id": "...", ...}
```

### 3. 設定後の確認

設定が完了したら、以下のコマンドで内容を確認できます（APIキーは表示されません）：

```bash
cat .env | grep -v "your_" | grep -v "^#"
```

## 注意事項

- `.env`ファイルには機密情報が含まれます。Gitにコミットしないでください（`.gitignore`に含まれています）
- APIキーは他人と共有しないでください
- サービスアカウントのJSONファイルも機密情報です。安全に保管してください
